ARDUINO / LINK DE DESCARGAR Y PASO A PASO

Para poder utilizar el Arduino se debe descargar este programa, si no lo tienes descargar aqui: //drive.google.com/drive/u/0/folders/16uY9RqEUHVykNgLEajLH_96z9968o5ZW

PASO A PASO

1. Descargar el archivo según el sistema operativo que tengas ya sea window o mac.

Una vez descargado el archivo, ejecutar como administrador e iniciar el proceso de instalación.

2. Aceptar cada una de las ventanas emergentes por que son contraladores que permitira que el programa funcione correctamente.

Una vez instalado abrir el programa y configurar en herramientas placa: "arduini uno"; y en el puerto: COM3(arduino uno)

¿PREPARADO PARA NUESTRA PRIMERA PROGRAMACIÓN?

Conectar los cables de la siguiente manera.

DESCARGAR LED CONTROL MASTER

https://cdn.discordapp.com/attachments/855882660059414538/855913878897426432/LedControl-master.zip
