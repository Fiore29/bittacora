ESCULPIENDO EN 3D CON ZBRUSH!

Utilizaremos un programa de diseño como illustrator o cualquier otro Software de tu preferencia y a partir de allí crear un boceto para un futuro modelado en el programa ZBrush, con el objetivo de poder tener líneas, tener vectores de la figura que queremos realizar, para esto necesitaremos dos vistas una frontal y otra lateral. Y a partir de estas vistas podremos crear el volumen del cuerpo. Aquí nos muestra cómo a partir de una esfera se puede crear un rostro, jalando y hundiendo polígonos.

Al iniciar el programa Z brite Es recomendable tener un diseño predeterminado para no empezar desde cero, nos podemos ayudar de illustrator u otros programas de diseño para crear un personaje y sobre este crear una figura con zBrush.

Una vez con el boceto listo cortamos la imagen y la exportamos en jpg para subirla al programa ZBrush y continuar con el esculpido. Con click derecho vamos creando volumen en el polígono y Con alt + click derecho vamos hundiendo ciertas zonas que que creamos convenientes para nuestro diseño.

COMANDOS

Click derecho: Puedes rotar la cámara del objeto.

Ctrl + click derecho: Puedes hacerle su objeto.

Alt + click derecho: Puedes mover la cámara del objeto

Letra W: Se activa el comando para escalar el objeto.

Letra X: Activar/Desactivar la asimetría.
