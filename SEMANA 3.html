En esta sesion aclaramos las dudas sobre que es el HTML y sus diversos. Tambien aprendi a como poner elementos dentro de otros elementos, esto se conoce como anidamiento.

La letra "p" se utiliza para especificar que es un parrafo, primero para abrir un parrafo se coloca los parentisis angulares y la palabra "p" dentro, para cerrar el parrafo que ya redactamos, se aplican los mismos simbolos pero en este caso antes de la "p" colocamos un "/", esto nos indica que nuestro parrafo esta cerrado de la manera correcta.

La letra "h" se utiliza para saber el tamaño de nuestros titulos, "h1" para titulos principales, "h2" para subtitulos, "h3" para pequeñas frases que acompañan al subtitulo, asi sucesivamente.

La palabra "strong" se utiliza para resaltar en el parrafo la palabra mas importante del texto o tambien podemos utilizarla para resaltar la palabra que dirige hacia otro destino. Para poder aplicar bien este elemento se tiene que poner primero "strong", luego la palabra que destaca del texto y cerramos nuevamente con "strong", recordar que el elemento siempre esta rodeado de parentesis angulares y para cerrar se coloca el "/".

Para agregar una imagen nueva a nuestro pagina web, primero debemos encontrar la imagen que necesitamos, una vez enconrada, procedemos a colocar el siguiente elemento "img", damos un "espacio" luego "src", seguimos con "=" ponemos "images/pic011.jpg", en este momento la computadora ya te da los enlaces predeterminados y solo damos click en lo que estamos buscando, para cerrar colocamos "alt="" /", recordar siempre que todos los elementos que ponemos deben estar dentro de los parentesis angulares.
